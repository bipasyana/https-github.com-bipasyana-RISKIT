Overview

The Dare Generator App is a fun and interactive tool designed to add excitement to group gatherings and parties. The app presents users with random dares, making it easy to break the ice and keep everyone entertained.

Getting Started

Prerequisites

  • Python Basics
  • Installing and Using Libraries
  • Tkinter for GUI

Installation

  1. Clone the repository: https://github.com/bipasyana/RISKIT.git
  2. Navigate to the project directory: cd Documents\RISKIT
  3. Install required packages: pip install tkinter random

Usage

  1. Start the App: Open the app to see the welcome screen.
  2. Generate a Dare: Click the "Generate" button to see a random dare.
  3. Complete the Dare: Click "Completed" to return to the main screen.

Code Structure

The code is organized into:

  • Imports and Data Initialization: Includes libraries and a list of dares.
  • Function Definitions: Manages showing and resetting dares.
  • GUI Layout: Sets up the welcome and dare display screens.

Dependencies

  • Python 3.x
  • Tkinter (pre-installed with Python)
  • Random module (part of Python's standard library)
